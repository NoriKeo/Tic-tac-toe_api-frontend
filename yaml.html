Tic Tac Toe API Dokumentation

Version: 1.0.0

Beschreibung: API für Login, Account-Erstellung, Passwort-Reset, Matches, Match-Historie und Score

POST /api/login – Spieler Login

Anfrage:

{
  "playerName": "string",
  "password": "string"
}

Antwort (200):

{
  "message": "string",
  "playerId": 0
}
POST /api/creataccount – Account erstellen

Anfrage:

{
  "playerName": "string",
  "password": "string",
  "securityAnswer": "string"
}

Antwort (200):

{
  "message": "string"
}
POST /api/newPassword – Passwort zurücksetzen

Anfrage:

{
  "playerName": "string",
  "password": "string",
  "securityAnswer": "string"
}

Antwort (200):

{
  "message": "string"
}
POST /api/matchHandler – Spielzug senden

Anfrage:

{
  "playerId": 0,
  "move": 0
}

Antwort (200):

{
  "message": "string",
  "matchID": 0,
  "move": 0,
  "computerMove": 0,
  "winner": 0,
  "score": [0, 0, 0],
  "playerPosition": [0, 1],
  "computerPlays": [2, 4]
}
POST /api/matchHistory – Match-Historie

Anfrage:

{
  "playerId": 0
}

Antwort (200):

[
  {
    "id": 1,
    "started_at": "timestamp",
    "ended_at": "timestamp",
    "verdict_id": 1,
    "playerMoves": [0, 1],
    "computerMoves": [2, 3]
  }
]
POST /api/score – Score abfragen

Anfrage:

{
  "playerId": 0
}

Antwort (200):

{
  "playerscore": 5,
  "computerScore": 3,
  "drawScore": 2
}
